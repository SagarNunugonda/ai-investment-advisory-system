# ai-investment-advisory-system
# AI-Powered Investment Advisory System

## Overview
Multi-agent AI system that provides stock recommendations for Indian markets using CrewAI and yFinance.

## Features
- Real-time NSE stock data
- Multi-agent reasoning
- Risk assessment
- Portfolio recommendations
- Dockerized deployment

## Tech Stack
- Python
- CrewAI
- yFinance
- Pandas
- Docker

## How to Run
1. pip install -r requirements.txt
2. python main.py
